# Block Elements

## Headers 見出し
Thêm dấu `#` vào đầu dòng để tạo bậc.
```
# Bậc 1
## Bậc 2
### Bậc 3
#### Bậc 4
##### Bậc 5
###### Bậc 6
```

```
| Tables        | Are           | Cool  |
| ------------- |:-------------:| -----:|
| col 3 is      | right-aligned | $1600 |
| col 2 is      | centered      |   $12 |

```
